# Kafka Consumer

Kafka consumer desenvolvida em **Spring Boot 3+**, responsável por consumir mensagens de um tópico Kafka.

---

## Tecnologias utilizadas

- Java 17
- Spring Boot 3+
- kafka
---


# Estrutura

## configuration
Configuração do listener Kafka (grupos, tópicos, factory).

## consumer
Classe que recebe e processa mensagens enviadas ao tópico.

## entity
Estrutura de dados utilizadas pelo consumidor.
